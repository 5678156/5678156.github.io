Fundemental code

  • cd

    • This is used to search deeper and open more files.

  • ls

    • This is used to see which new files and directories you can open.

  • pwd

    • This is used to help keep track of where you are,so you don't get lost.

  • mkdir

    • This is used to create and new directory

  • rm

    • This is used to remove a file or directory

  • cp

    • This is used to copy files between directories

  • mv

    • This is used to move or rename files and directories

  • grep

    • This is used to search for specific text within files

  • less

    • This is used to view to contents of large files
