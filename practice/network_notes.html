Notes about the internet

Vocabulary

  • Protocol -A set of rules that defines how data is transmitted and exchanged between devices, ensuring they can communicate effectively
  • Packet -A small, formatted unit of data that travels across a computer network
  • Header -A supplemental data placed at the beginning of a block of data to provide information about the content that follows
  • Payload -The actual data being transmited or the essential part of a message, seperate from the accompanying headers or metadata
  • Router -A hardware device that forwards data packets between computer networks, typically connecting a home or office local area netwrok to the internet
  • Routing Table -A data table stored in a computer or router that lists rules, or routes, for directing data packets to their destination on a network
